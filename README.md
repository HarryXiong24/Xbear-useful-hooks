### 🏠 [Homepage](https://github.com/HarryXiong24/Xbear-useful-hooks)

## Install

```sh
yarn install
```

## Run tests

```sh
yarn

yarn dev
```

## Run build

```sh
yarn

yarn build
```

## Run lint

```sh
yarn lint
```
